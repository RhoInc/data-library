This collection of datasets emulates data used in clinical data cleaning and validation, as described in this article. Each dataset drives one of Rho's clinical data cleaning graphics:
  • forms: crf-heat-map, which visualizes the completeness of study data collection by a nested selection of groupings
  • queries: query-overview, which visualizes a study's data queries by group and query status
  • visits: participant-visit-listing, which visualizes the schedule of events by participant
